Synthese - Decision Tree + selection Personnel

Le modele Decision Tree avec selection Personnel a ete optimise avec le F1-score comme metrique principale.
Les meilleurs parametres sont : criterion=gini, max_depth=3, min_samples_leaf=1, min_samples_split=2.
Accuracy train : `0.7848`, accuracy test : `0.8046`. Les scores train et test sont tres proches, ce qui indique que l'overfitting est limite.
F1-score test : `0.7385`, precision : `0.7500`, recall : `0.7273`. Le recall est correct : le modele retrouve une bonne partie des survivants.
F1-score moyen en validation croisee : `0.6978` avec un ecart-type de `0.0418`. L'ecart-type du F1-score en validation croisee est plus eleve, ce qui montre une stabilite moins reguliere selon les folds.
La matrice de confusion donne TN=138, FP=24, FN=27, TP=72.

Points a retenir :
- Metrique principale : F1-score.
- Le score d'accuracy est utilise comme indicateur secondaire.
- Le recall indique la capacite a retrouver les survivants.
- La custom accuracy accepte les faux negatifs selon le choix d'interpretation du projet.
