Synthese - KNN + selection Personnel

Le modele KNN avec selection Personnel a ete optimise avec le F1-score comme metrique principale.
Les meilleurs parametres sont : metric=euclidean, n_neighbors=11, weights=distance.
Accuracy train : `0.9760`, accuracy test : `0.7778`. L'ecart entre le train et le test est important, ce qui indique un overfitting marque.
F1-score test : `0.7041`, precision : `0.7113`, recall : `0.6970`. Le recall est moyen : le modele retrouve une partie importante des survivants, mais en manque encore plusieurs.
F1-score moyen en validation croisee : `0.6637` avec un ecart-type de `0.0254`. L'ecart-type du F1-score en validation croisee est faible, ce qui indique une stabilite correcte entre les folds.
La matrice de confusion donne TN=134, FP=28, FN=30, TP=69.

Points a retenir :
- Metrique principale : F1-score.
- Le score d'accuracy est utilise comme indicateur secondaire.
- Le recall indique la capacite a retrouver les survivants.
- La custom accuracy accepte les faux negatifs selon le choix d'interpretation du projet.
